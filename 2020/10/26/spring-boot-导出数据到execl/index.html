spring boot 导出数据到execl

发表于 2020-10-26 更新于 2020-10-28 分类于 springboot
平时开发到时候可能会遇到需要将数据导出到excel中，或者从excel中读取数据进行批量操作，这时候我们可以使用appache的poi来进行操作，也可以使用阿里巴巴的easyexcel来操作，这里以后者为例进行介绍

然后我们需要新建一个类用于和excel各字段相互对应，叫做模板，并且该类要继承自com.alibaba.excel.metadata.BaseRowModel.

每个字段上都需要加上@ExcelProperty注解，其中value属性表示execl中的列名称，index表示excel中的列号，下标从0开始。而且像这种方法应该以工具类的形式提供出来给各业务使用

可以看到实际还是使用了前面提供的easyexcel单sheet导入的方法，只不过在导入后对读取到的数据进行了一次校验，这样做的好处就是可以让前端在导入的时候明确的知道每一条数据导入成功与否，以及他失败的原因，校验方法是由具体的子类去实现的。
