使用hexo+github搭建自己的博客

发表于 2020-10-20 更新于 2020-10-21
利用hexo+github快速搭建博客，再也不用担心想写的笔记找不到地方存放了，还是能随时随处查看，简直不要太方便！

安装环境

​ hexo需要nodeJs环境支持，如果同时需要将自己的内容托管到github的话，还需要安装git，假设已经安装好nodeJs和git。一般我们使用淘宝镜像下载nodejs包

搭建博客

第一步

​ 新建一个文件夹，用于存放所有关于博客的文件，后面的所有操作都是在该文件夹下进行的，假设名称就是Blog

第二步

​ 进入该文件夹，然后通过cmd执行hexo init,对该文件夹进行hexo初始化，也可以右键单击Blog,选择gitBash打开，然后执行该命令，然后继续执行npm install –save hexo-deployer-git命令，添加hexo对git的支持，以便支持后续上传到github

新建博客

第一步

以后通过hexo new命令的时候会在文章同级目录下生成一个同名的文件夹，这里就可以存放静态资源比如图片，然后在Blog目录下安装插件npm install https://github.com/CodeFalling/hexo-asset-image --save,然后就可以上传图片了

第二步

设置文章显示摘要

如果想设置文章只显示摘要,不显示全部，有两种办法

  • 在文章抬头处加上description:,将摘要内容写在这里，则只会显示摘要
  • 在md中，在想要截断处加上<!--more-->,则只会显示该行之前的内容

常用命令：

  • hexo clean 清理缓存
  • hexo g 生成
